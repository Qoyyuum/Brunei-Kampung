Brunei Kampung JSON

This is just a simple JSON endpoint for Brunei Kampung, and Mukim

JSON file: brunei_kampung.json

CSV file: brunei_postcodes

How to use:

There are no other methods than GET.

HTTP Response

GET /brunei_kampung.json HTTP/1.1
Host: brunei-kampung.netlify.com
Accept: */*
Cache-Control: no-cache
Host: brunei-kampung.netlify.com
Accept-Encoding: gzip, deflate
Connection: keep-alive
cache-control: no-cache

With cURL

curl -X GET \
https://brunei-kampung.netlify.com/brunei_kampung.json \
-H 'Host: brunei-kampung.netlify.com' \
-H 'cache-control: no-cache'
